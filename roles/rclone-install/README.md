# Rclone Install Role

Ansible роль для установки rclone через скачивание бинарника с официального сайта.

## Особенности

- Поддержка архитектур: amd64, arm64, arm, 386
- Автоматическое определение архитектуры системы
- Скачивание актуальной версии rclone
- Установка в `/usr/local/bin`
- Проверка установки

## Переменные

- `rclone_version`: версия rclone для установки (по умолчанию: v1.71.0)
- `rclone_install_dir`: директория установки (по умолчанию: /usr/local/bin)
- `rclone_base_url`: базовый URL для скачивания

## Использование

```yaml
- hosts: servers
  become: true
  roles:
    - rclone-install
```

## Требования

- Ansible 2.9+
- Доступ к интернету для скачивания rclone
- Права sudo для установки
